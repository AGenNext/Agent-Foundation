# Ecosystem Pledge

AGenNext cannot enforce the world.

AGenNext can build, support, recommend, recognize, certify, publish, and sustain an open ecosystem around its foundation, platform, products, partners, providers, and community.

## Core pledge

```text
We will build and support the ecosystem.
We will recommend clear practices.
We will recognize trusted work.
We will publish evidence-backed artifacts.
We will protect human accountability.
We will keep participation open and trust guarded.
```

## What we commit to do

AGenNext commits to:

```text
maintain open repositories
publish shared contracts
support community contribution
review public proposals
recognize compatible implementations
publish reference architectures
support partners and providers
protect provenance and evidence
maintain human accountability gates
preserve the foundation's coherence
```

## What we do not claim

AGenNext does not claim to control:

```text
the internet
external forks
external vendors
external communities
external deployments
independent implementations
```

## Trust model

```text
Anyone may participate.
Not everything becomes trusted.
```

Trusted recognition requires:

```text
evidence
review
compatibility
provenance
human accountability
published decision record where needed
```

## Ecosystem support paths

AGenNext will support the ecosystem through:

```text
GitHub issues
GitHub discussions
GitHub projects
GitHub actions
public documentation
reference implementations
community loops
partner/provider guidance
trusted artifact publishing
release notes
```

## Recognition levels

```text
Proposed
Compatible
Recommended
Certified
Supported
Deprecated
Untrusted
```

## One-line pledge

AGenNext pledges to build and support an open, federated, evidence-backed ecosystem where anyone may participate, trusted work is recognized through clear gates, and human accountability remains at the center.
