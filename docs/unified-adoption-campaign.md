# Unified Adoption Campaign

Status: active campaign package.

## Campaign name

```text
AGenNext Unified Agent Adoption Campaign
```

## Core message

```text
Proto-first.
Conformance-checked.
Human-accountable.
Cloud-native.
Open ecosystem.
```

## Purpose

Unify adoption across the AGenNext ecosystem by aligning contributors, maintainers, partners, providers, builders, and customers around one governed operating model.

## Adoption thesis

```text
Agents need shared language.
Shared language needs contracts.
Contracts need conformance.
Conformance needs evidence.
Evidence needs accountability.
Accountability needs humans at the gate.
```
