# CNCF Maintainer Path

CNCF can be an eventual neutral home, maintainer model, or governance reference for Agent-Foundation if the project matures into a broadly adopted cloud-native foundation layer.

This document is aspirational. It does not claim CNCF acceptance, endorsement, ownership, or maintainership.

## Core rule

```text
AGenNext stewards now.
CNCF-style governance is a future path, not a current claim.
```

## Why CNCF is relevant

Agent-Foundation and Agent-Fabric align with cloud-native operating principles:

```text
open source
community contribution
neutral governance
reconciliation loops
Kubernetes-native operation
service mesh observability
supply-chain provenance
maintainer-led releases
interop and portability
```

## Current state

```text
Steward: AGenNext
Founder-origin: Chinmay Panda
Repository: AGenNext/Agent-Foundation
Governance: founder-originated, community-contributed, maintainer-curated
Publication: human accountability gate
Trust: evidence, provenance, CI, review, release gates
```

## Future path

A future CNCF-style path may include:

```text
public roadmap
clear maintainers
open contribution process
documented governance
neutral trademark/IP posture
security policy
release process
artifact provenance
adoption evidence
interoperability proof
multiple contributors/organizations
community meetings
technical oversight readiness
```

## Donation or external maintainership criteria

Before proposing CNCF donation or external neutral maintainership, the project should demonstrate:

```text
working implementation
active community
multiple maintainers
clear scope
clear license
security policy
release history
production or serious adoption signals
documented governance
compatibility and conformance tests
non-overlap with existing CNCF projects or clear integration value
```

## Governance transition model

```text
AGenNext stewarded
  -> community contributed
  -> multi-maintainer
  -> neutral governance proposal
  -> foundation review
  -> accepted neutral home if appropriate
```

## What does not change

Even if a neutral foundation path is pursued:

```text
No direct Fabric alteration
No artifact without provenance
No release without human accountability
No hidden compatibility break
No trusted publication without review
```

## One-line definition

CNCF can be a future neutral governance and maintainer path for Agent-Foundation, but today AGenNext remains the steward while the project builds the openness, evidence, adoption, and governance maturity required for such a transition.
